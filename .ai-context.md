# Project AI Context

> **Overview:** The Harvesters Reporting System is a standalone, role-based web application for Harvesters International Christian Center. It standardizes report submission, review, approval, and analytics across all campuses and groups, replacing fragmented manual processes with a single, auditable, and analytics-ready system. The platform is built for future federation into a broader CRM, with strict type safety, config-driven UI, and a modern design system. The stack is Next.js 16+, TypeScript strict, Ant Design v6, Tailwind CSS v4, Prisma, and PostgreSQL.
